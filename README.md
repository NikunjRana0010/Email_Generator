# Email Generator

AI-powered email generator built using OpenAI API.

## Features

* Generate professional emails
* Generate customer support emails
* Generate follow-up emails
* Easy prompt-based interface

## Technologies Used

* Python
* OpenAI API
* Requests

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python Email_writer.py
```

## Example

### Input

```text
Write a follow-up email for pending payment.
```

### Output

```text
Professional follow-up email generated by AI.
```
